HTML5 Test
==

Simple React Application

# Usage:

Clone this project, then install dependencies
```
git clone git@github.com:sangdth/html5-test.git
cd html5-test/
yarn install
yarn start
```

Go to `http://localhost:3000` (or the suggested port if 3000 is used).
